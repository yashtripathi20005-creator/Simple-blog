# Simple Blog Layout (Static HTML/CSS)

A minimal, clean blog template built with Python and pure CSS.  
Just open `index.html` in your browser – no server, no build step.

## Files
- `index.html` – the complete blog page (self‑contained).
- `generate_blog.py` – Python script that creates the HTML (run it to regenerate).

## Quick start
1. Save the HTML file and open it.  
2. Or run `python generate_blog.py` to rebuild `index.html` from the Python template.
